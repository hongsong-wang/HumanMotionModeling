# [面向人与数字人的动作-语义建模: 动作理解与生成](https://github.com/hongsong-wang/HumanMotionModeling/blob/main/%E5%8A%A8%E4%BD%9C%E7%90%86%E8%A7%A3%E4%B8%8E%E7%94%9F%E6%88%90.pdf) （[CSIG云讲堂报告2025-05-12](https://mp.weixin.qq.com/s/j-1wJYhhkkWRAKt7IyYg_w)），[百度网盘PPT链接](https://pan.baidu.com/s/14uMWSvBXfjMv4kXJIpiCDw?pwd=56qt)

# 报告摘要
在生成式人工智能快速发展的背景下，人体动作建模正从传统受限场景下的判别式行为识别，逐步发展为开放环境中的动作-语义双向建模与跨模态动作生成，从而实现动作理解与生成的一体化发展。这一进展打通了视频监控场景下人体动作理解与网络空间中数字人的动作生成与驱动之间的关键链路。本报告将介绍人体动作理解与生成的前沿研究进展，并汇报报告人近几年的相关工作，主要包括开放环境下的动作理解、文本驱动的动作生成以及音频驱动的动作生成等。
